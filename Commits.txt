Guia rápido de Git (enxuto)

# Clonar
git clone <URL-do-repositório>
cd <nome-do-projeto>

# Branching (opcional, se usar develop)
git checkout -b feature/<nome>

# Commitar mudanças
git add .
git commit -m "feat: descrição curta da mudança"

# Enviar
git push -u origin feature/<nome>

# Dicas
- Não commitar .env (já está no .gitignore)
- Commits pequenos e temáticos
