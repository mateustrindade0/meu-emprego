# Meu Emprego – Sistema de Cadastro de Candidaturas

## Ideia Inicial
Desenvolver uma aplicação desktop em Python com a biblioteca Tkinter que permita registrar, visualizar e analisar candidaturas de emprego. O usuário pode cadastrar novas vagas com informações como empresa, cargo, data, tipo da vaga e status.

## Propósito do Repositório
Este repositório hospeda o código-fonte da aplicação “Meu Emprego”, bem como os artefatos de entrega (como o arquivo de dados CSV e as capturas de tela). O objetivo é fornecer uma solução funcional e simples para gerenciamento de candidaturas, com persistência de dados e análise gráfica.
